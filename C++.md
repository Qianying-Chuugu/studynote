### 1. 常量：记录程序中不可更改的数据
#### a.#define 宏常量 常量值；  
在main()函数外定义宏常量。
#### b.const 数据类型 常量名 = 常量值;

#### a.char 变量名[] = "字符串内容";
#### b.string 变量名 = "字符串内容";
使用b的方式定义字符串变量，需要引入string头文件。  
### 3. 输入对象cin
#### cin>>变量名; 即可从键盘输入数据
注意：

cin>>变量名1>>变量名2; 能够连续给多个变量录入数据，任何数据类型都能这样录入。cin本身的录入逻辑是会忽略开头的所有空白符（空格、回车、Tab），直到遇到有效字符，再读取到下一个空白符为止。
### 4. 输出对象cout
#### cout<<变量名; 输出变量名所代表的数据
注意：

cout<<变量名1<<变量名2; 输出多个变量所代表的数据，任何数据类型都能这样输出。通常结合endl使用，表示输出一个换行符,即：cout<<变量名1<<endl<<变量名2;

### 5. 运算符
a. /除法  
对于整数，除法结果为整数(向零取整)，对于浮点数，除法结果为浮点数。  
b. %取余

### 指针：储存内存地址的变量
#### const修饰指针
1. 常量指针：  
   const int *p;  
   常量指针的本身存储的地址值可以改，但地址值所对应的数据不能通过这个指针改。
2. 指针常量：  
   int *const p;
   指针常量的本身存储的地址值不能改，但地址值所对应的数据可以通过这个指针改。
3. 指针常量指针：  
   const int *const p;
   指针本身存储的地址值和地址值所对应数据不能通过这个指针改。

## 内存分区模型

C++在程序运行时，操作系统把物理内存或虚拟地址空间划分为不同的区域进行管理，通常划分为以下几个主要分区：
1. 代码区：存放程序代码转化成的二进制机器指令，通常是只读的(频繁访问的程序，内存中只分配一块代码)且共享的(防止程序修改了它的指令)。 如函数定义、变量声明、类定义等。  
   
2. 数据区：存放已经初始化的数据，如全局变量、静态变量、常量(字符串常量、const修饰的变量)等。也存放未初始化的数据，如未初始化的全局变量等，该区域的数据在程序结束后由操作系统释放。  

3. 堆区： 由程序员手动申请和释放的内存，从低地址到高地址增长，若最终内存未释放，则程序运行结束，该内存空间就会回收。

4. 栈区：存放函数的局部变量、函数参数、返回地址等，由系统自动分配和释放，从高地址到低地址增长。
