# Python-Base
## 基础内容
1. 输出函数print(*values,sep,end)  

values可以是任意个任意数据类型变量或正文内容
sep=""表示不同的value的间隔符  
end=""表示该输出的结束符  

2. 数据类型  

基础：整型int,浮点型float,布尔型bool,字符串str   
容器：列表list,元组tuple,字典dict,字典dict  
type() 判断数据类型

3. 操作符

- // 整除反整  
  ** 幂
- and 与  
  or 或  
  not 非

4. 字符串  
- value能用f格式化输出:f"正文{变量名}正文"  
槽{}内部控制变量格式:{变量名:}  
``` 
格式	        效果
.2f	        保留 2 位小数
05d	        整数补 0 到 5 位
.1%	        百分比 1 位小数
<5, >5	    左/右对齐，占 5 位
^5	        居中
,	        千位分隔符
b/o/x	    二/八/十六进制
```
- 字符串之间能直接用+相加拼接
- 字符串片选str1[0:-1]->选取除了尾字符的所有字符，选取位置是前闭后开的
- 字符串有大量方便的内置函数，如：  
  len(x)     返回x的长度  
  str(x)     返回x的字符串形式   
  chr(x)     返回ASCII编码x对应的字符  
  ord(x)     返回字符x对应的ASCII编码

5. 输入函数input(prompt)  
prompt可以写输入前的文字提示

6. if判断结构  

(python是基于对齐划分语句结构的语言)
``` 
if 条件表达式1：
    语句块1
elif 条件表达式2：
    语句块2
·······
else 条件表达式：
    语句块
```
7. for迭代循环结构
``` 
for 任意变量 in 可循环体：
    语句块1
else:
    语句块2
```
- 可循环体：字符串,列表,元组,range,字典,集合  
  一般的，用range()生成数字序列：
  range(start,end,step)
  start-end表示的范围是前闭后开的
- continue和break关键字适用
- else是在循环正常结束时可以调用执行,break出循环不会调用
8. while循环结构
   
```
while 循环条件：
    语句块1
else:
    语句块2
```
9. match匹配结构
```
match 待匹配变量:  
    case 模式1:  
        匹配成功执行  
    case 模式2:  
        匹配成功执行  
    case _:  
        所有模式都不匹配时执行
```
类似于其他语言的switch,但是不穿透。  
待匹配变量可以支持列表,元组,字典等
- 可以匹配值,能通过a|b的方式多值匹配
- 匹配还可以带条件,条件满足才能执行(x if x>0)
- 还能匹配数据类型,如int(),str(),list()等
- 字典和序列匹配在此暂略

10. 函数基础

``` 
def 函数名(形参1，形参2···)
    函数体
    return 返回值

调用：函数名(实参1，实参2···)
```
- 默认参数规则与C相同
- return 能返回多个值，本质返回个元组
- 调用时可以直接指定形参传参,不用顾虑顺序：
  函数名(形参1=实参1,形参2=实参2···)
- 可变参数：
```
参数书写顺序：

固定参数-*args-默认参数=*kwargs

1. *args
该参数会接受任意多个无名实参，在函数内部组合成一个元组
2. **kwargs
该参数会接受任意多个key=value实参，在函数内部组合成一个字典

反过来，对列表/元组使用*能解包成多个无名参数；对字典使用**能解包成多个key=value参数
```
