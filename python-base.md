# Python-Base
## 基础内容
1. 输出函数print(*values,sep,end)  

values可以是任意个任意数据类型变量或正文内容
sep=""表示不同的value的间隔符  
end=""表示该输出的结束符  

2. 数据类型  

基础：整型int,浮点型float,布尔型bool,字符串str   
容器：列表list,元组tuple,字典dict,字典dict  
type() 判断数据类型

3. 操作符

- // 整除反整  
  ** 幂
- and 与  
  or 或  
  not 非

4. 字符串  
- value能用f格式化输出:f"正文{变量名}正文"  
槽{}内部控制变量格式:{变量名:}  
``` 
格式	        效果
.2f	        保留 2 位小数
05d	        整数补 0 到 5 位
.1%	        百分比 1 位小数
<5, >5	    左/右对齐，占 5 位
^5	        居中
,	        千位分隔符
b/o/x	    二/八/十六进制
```
- 字符串之间能直接用+相加拼接
- 字符串片选str1[0:-1]->选取除了尾字符的所有字符，选取位置是前闭后开的
- 字符串有大量方便的内置函数，如：  
  len(x)     返回x的长度  
  str(x)     返回x的字符串形式   
  chr(x)     返回ASCII编码x对应的字符  
  ord(x)     返回字符x对应的ASCII编码

5. 输入函数input(prompt)  
prompt可以写输入前的文字提示

6. if判断结构  

(python是基于对齐划分语句结构的语言)
``` 
if 条件表达式1：
    语句块1
elif 条件表达式2：
    语句块2
·······
else 条件表达式：
    语句块
```
7. for迭代循环结构
``` 
for 任意变量 in 可循环体：
    语句块1
else:
    语句块2
```
- 可循环体：字符串,列表,元组,range,字典,集合  
  一般的，用range()生成数字序列：
  range(start,end,step)
  start-end表示的范围是前闭后开的
- continue和break关键字适用
- else是在循环正常结束时可以调用执行,break出循环不会调用
8. while循环结构
   
```
while 循环条件：
    语句块1
else:
    语句块2
```
9. match匹配结构
```
match 待匹配变量:  
    case 模式1:  
        匹配成功执行  
    case 模式2:  
        匹配成功执行  
    case _:  
        所有模式都不匹配时执行
```
类似于其他语言的switch,但是不穿透。  
待匹配变量可以支持列表,元组,字典等
- 可以匹配值,能通过a|b的方式多值匹配
- 匹配还可以带条件,条件满足才能执行(x if x>0)
- 还能匹配数据类型,如int(),str(),list()等
- 字典和序列匹配在此暂略

10. 函数基础

``` 
def 函数名(形参1，形参2···):
    函数体
    return 返回值

调用：函数名(实参1，实参2···)
```
- 默认参数规则与C相同
- return 能返回多个值，本质返回个元组
- 调用时可以直接指定形参传参,不用顾虑顺序：
  函数名(形参1=实参1,形参2=实参2···)
- 可变参数：
```
参数书写顺序：

固定参数-*args-默认参数=*kwargs

1. *args
该参数会接受任意多个无名实参，在函数内部组合成一个元组
2. **kwargs
该参数会接受任意多个key=value实参，在函数内部组合成一个字典

反过来，对列表/元组使用*能解包成多个无名参数；对字典使用**能解包成多个key=value参数
```
  
## 容器(组合数据类型)
所有的组合数据类型，本质上是个对一块内存的引用，本身储存的是内存地址
### 序列容器
通用操作：  
1. 索引取值[i]
2. 切片[start:end:step]
3. 长度函数len()
4. 成员判断  
   数据 in/not in 序列
5. 拼接+
6. 重复*  
   序列*=3 ->表示序列自身复制三份拼接在一起
7. 可做循环体for
8. 支持max()和min()  
   max(序列) ->返回最大值,字符串按ASCII码比较
9. 支持.index()和.count()  
    序列.index(a) ->返回序列中第一个a出现位置  
    序列.count(a) ->统计序列中出现a的次数
10. 支持sum()
#### 列表list[]
特性:  
元素有序、可增删、可重复、可迭代、可嵌套  
定义:  
通过[]或者list()创建  
修改:  
1. 索引修改： L[1]=1
2. 片选修改： L[1:3]=[3,4]  

添加:  
1. L.extend(X) ->末尾加入可迭代对象X(拆分后依次加入)
2. L.append(X) ->末尾加入元素X
3. L.insert(a,X) ->在索引a位置处插入元素X

删除:  

1. del L[索引] ->按索引或片选删除元素
2. L.pop[索引] ->弹出返回索引处元素,默认指定末尾元素
3. L.remove(元素) ->按值删除第一个出现的元素
4. L.clear()  ->清空列表

内置函数：
1. L.sort()  ->原地升序排序  
   L.sort(reverse=True) ->原地降序排序
2. L.reverse()  ->原地反转列表
3. sorted(列表) ->返回新生成的升序排序列表
4. L.copy() ->返回复制的新列表

列表推导式：  
基本格式    
`[表达式 for 变量 in 可迭代对象] `  
带筛选条件  
`[表达式 for 变量 in 可迭代对象 if 条件]`  
带elif条件    
`[表达式1 if 条件 else 表达式2 for 变量 in 可迭代对象]`  
for嵌套(先外后内)  
`[表达式 for 变量1 in 可迭代对象1 for 变量2 in 可迭代对象2]`

#### 字符串str“”
特性：  
元素于定义阶段确定后，原串元素不能修改、增加、删除，元素有序  
定义：  
s=''/""/""" """  
内置函数：  
包含转换大小写、查找元素或字串、分割串等操作，不赘述，需要可以自行查阅,然后补充在此  

字符串格式化：  
1. f-string格式化(最新)
2. str.format () 格式化
3. 旧式 % 格式化  

转义字符处理：  
在字符串前加r/R，关闭转义功能，字符串中的\会被当作普通字符处理，常用于文件路径、正则表达式。

#### 元组tuple()
特性：  
元素于定义阶段确定后，原串元素不能修改、增加、删除，元素有序(如果有list列表，list的内容是可以修改的)  
定义：  
通过()或者tuple()创建   
单元素元组定义t=(1,) 

### 集合容器set{}
特性：  
内部元素具有唯一性(自动去重)、无序性、支持增删元素、可迭代性，所存储元素必须是不可变类型的   
定义：  
1. s={1,2,3} 括号里在定义时必须有元素  
2. s=set()  
3. s={} 这是空字典，不是空集合

增加：  
1. s.add(x) ->元素x加入集合
2. s.update(x) ->可迭代对象x拆分后加入集合

删除：  
1. s.remove(x) ->删除元素x(不存在会报错)
2. s.discard(x) ->删除元素x(不报错)
3. s.pop() ->随机删除并返回一个元素
4. s.clear() ->清空集合

集合运算：  
取并集|，交集&，差集-，对称差^

集合关系判断：  
<、==、>、<=、>=等关系比较就是在比较两个集合元素的包含关系

不可变集合：  
创建的该类集合元素也不能增加删除了。  
`s=frozenset()`

集合推导式:  
规则和列表推导式一致。  

### 字典容器dict{}
特性：  
字典是Python中唯一的标准映射类型，字典的基本单位是键值对（key: value），一个 key 对应唯一的 value。支持对字典元素的增删改。  
定义：    
d={key1:value1,key2:value2} or d=dict()  
其中key必须是不可变类型，value可以是任意类型  
重复的key，后面的value会覆盖前面已有的

查找：  
1. d[key] ->返回key对应的键值  
2. get(key, 默认值) ->未找到键值则返回默认值
3. d.keys(),d.values(),d.items()  
   直接以列表形式返回所有key/value/键值对(对用元组承接)

修改：  
1. d[key]=newvalue ->修改key对应的value
2. d.update(字典) ->原本已有的key对应修改value,没有的key则创建

增加：  
1. d[key]=newvalue ->没有的key会添加
2. d.setdefault(key, 默认值) ->
key 存在时，返回原有 value，不修改字典；key 不存在时，新增键值对并返回默认值

删除：  
1. del d[key]  ->删除指定键值对
2. d.pop(key)  ->删除键值对并返回value
3. d.popitem() ->删除最后加入的键值对，返回键值对元组
4. d.clear()  ->清空字典
5. del d   ->删除字典变量

字典推导式：  
`{key表达式: value表达式 for 变量 in 可迭代对象 if 条件}`

字典解包**：  
1. 函数传参(对应之前讲的**kwargs可变关键字参数)
2. 字典合并
   d={**d1,**d2} 或 d=d1 | d2

### 容器转化关系
1. 列表元组互转  
   list()和tuple()直接互转，无损失
2. 列表元组转字符串  
   “A”.join(可迭代对象)    
   可以把可迭代对象里的所有元素按照拼接符A拼成一个字符串，要求元素必须是字符串  
   如果不是字符串，用生成式先转化为字符串  
   .join(str(num) for num in nums)
3. 字符串转列表元组  
   直接list()/tuple()只能把字符串每个字符作为一个元素拆分 
   字符串.split(A)  
   把字符串按字符A为间隔划分元素组成列表
4. 序列类和集合互转  
   序列转集合，要求元素必须是不可改类型  
   两者互转用构造函数set()/list()等即可
5. 其他类型转字典  
   传入dict()的参数必须是元素的长度都是2的可迭代对象，前者将做key，后者做value  
   也能用zip(A,B)组合做参，A的元素为key，B的元素为value
6. 字典转其他类型  
   str()是全部字符转化
   set()只支持key的转化
   其他的默认处理key，需要利用.values()/.items()
